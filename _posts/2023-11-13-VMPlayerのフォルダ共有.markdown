# フォルダ共有の手順

## 前提

- 共有するフォルダ名はwork

1. マシンの設定にて、右のタブからフォルダを共有
1. 以下をマシンにて打鍵

```sh
echo user_allow_other >> /etc/fuse.conf
echo '.host:/ /mnt/hgfs fuse.vmhgfs-fuse allow_other 0 0' >> /etc/fstab

mkdir -p /mnt/hgfs
vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other
ln -s /mnt/hgfs/work /work
```
