ファイル一括リネーム

```sh
rename aaa_ bbb_ *.txt
rename <before> <after> <target file(s)>
```

### example util-linux / BSD rename

```
$ touch aaa_{1..3}.txt
$ rename aaa_ bbb_ *.txt
$ ls -1
bbb_1.txt
bbb_2.txt
bbb_3.txt
$ 
```

Perl版 renameもあるらしく、`rename 's/old/new/' *.txt`で置換できるらしい。
